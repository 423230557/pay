Caomao_2018_Beta_v1.0
===============

一键聚合支付系统，支持第三/四方代码接入，一个支付API接入，轻松实现收款。
目前整合包括：

 + 微信
  + 微信扫码
 + 支付宝
  + 支付宝web
   + 支付宝wap
    + 支付宝扫码
 + QQ
  + QQ扫码
 + 更多等待...

> 运行环境要求PHP > 5.6以上(推荐7.0.*)。

使用系统前请：
>详细开发文档参考 [ThinkPHP5完全开发手册](http://www.kancloud.cn/manual/thinkphp5)
